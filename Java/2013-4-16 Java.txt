\65289　文字コード

カッコの中が最初に計算される
()の中は"で囲っていない場合は数字となる。文字が入っているとエラーとなる。


変数の定義前に出力すると
シンボルを見つけられませんとエラーが出る。

変数の初期化前に出力しようとすると
変数xは初期化されていない可能性がありますと出る。

int x;
x = 10;
int x;
こんな風に書くと変数xはすでに定義されていますと出る。

変数名のルール
１．記号は使えない（$と_は使える）
２．先頭に数字は使えない。
３．予約語は使えない。
４．予約語でも一文字でも大文字にすると予約語ではなくなる。
