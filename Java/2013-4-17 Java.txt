小テストについて
基本型の定義について
char c = 97;　コンパイルも実行も可能！
しかし、出力は97ではなくaとなる！
aの文字コードが97だから！
char型に文字を入れると、１文字としてではなく文字コード（正の整数）として格納される。

int a = 'A';　コンパイルも実行も可能！
''で囲むと囲んだ文字が文字コードとして代入されるから！


エスケープシーケンスを表すには\またはバックスラッシュ

byte型は-127〜127の値のみ表示できる。
1バイトは8ビットだから255種類のパターンがある！！

int i = 97;
char c = i;
はコンパイルできない。
int型とchar型では扱える情報の量が違うから！！
intは32bit charは16bit

授業
比較演算子はboolean型の結果を返す。
ifの()の中はboolean型の結果を返すなら何でもオッケー！！

|| &&　ショートサーキット（短絡評価）演算子

リファクタリング
プログラムの実行結果は変わらないように中身を変えていくこと！

switch文の()の中に入れられる型は以下！！
byte short int char 
Enum (Java5以降)
String (Java7以降)

String line = reader.readLine();
char c = line.charaAt(x)
標準入力した文字列のx番目の文字列をchar型のcに代入できる。
